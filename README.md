# Bridal-prep-app

A bridal prep app that turns “I have no idea what I’m doing” into “I’m confident, prepared, and excited” by helping brides discover their style, understand their budget, and know exactly what to expect at their dress appointment.

## MVP

- Style quiz
- Budget guide
- Appointment checklist
- Confidence prep

## Tech Stack

React • Node • Express • PostgreSQL • Jest
